Ansible Role: mxout
=========

This installs and configures Postfix on RHEL machines.

Requirements
------------

A fresh RHEL machine.

Role Variables
--------------

The next hop is declared through variables to the role. Default values are available in the role.

```
nexthop: defaut.example.org
nexthop_port: 25
```
